## 使用 picgo+github 搭建图床

### 下载安装 picgo

[PicGo](https://picgo.github.io/PicGo-Doc/zh/)

### 新建一个仓库并获取token

1. 新建仓库或者使用已有的仓库
2. 在右上角点击个人头像，进入 <kbd>settings</kbd> 页面，或直接点击 [添加token](https://github.com/settings/tokens)
3. 点击 <kbd>Generate new token</kbd> 生成新的token即可，注意要给 ==token== 权限才行。

### 配置picgo

1. 打开picgo配置页面，在 <kbd>图床设置</kbd> 页面选择  <kbd>GitHUb</kbd>
2. 按照 placeholdText 的提示一一填写配置信息即可
